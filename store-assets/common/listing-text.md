## Short Description

Discipline RPG habit tracker with streaks, quests, focus sessions, and XP progression.

## Full Description

Nithya is a discipline-focused productivity app built like an RPG.

- Build daily habits and complete quests.
- Track streaks and progress with XP and ranks.
- Use focus sessions to improve concentration.
- See insights, analytics, punishments, recovery plans, and achievement history.
- Enable reminders to stay consistent without notification spam.

Nithya is designed for users who want structure, momentum, and clear progress loops for self-improvement.
