# Aptoide Checklist

## Required

1. Package name: `com.rohith.nithya`
2. Keep same signing key for updates.
3. Upload signed APK (`app-release-signed-v1.0.apk`).
4. Set age rating: `13+`.
5. Provide privacy policy URL: `https://nithya.fit/privacy-policy.html`.
6. Add icon, screenshots, and listing text from `../common/listing-text.md`.

## Trust and Safety

1. Publish the matching `SHA256SUMS.txt` with each release.
2. Link users only to your official Aptoide app page and GitHub release page.
