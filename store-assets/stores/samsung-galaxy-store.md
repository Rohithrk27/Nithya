# Samsung Galaxy Store Checklist

## Required

1. Package name: `com.rohith.nithya`
2. Keep same signing key across updates.
3. Upload signed APK (`app-release-signed-v1.0.apk`).
4. Set content/age rating: `13+`.
5. Provide privacy policy URL: `https://nithya.fit/privacy-policy.html`.
6. Add icon + screenshots + listing copy from `../common/listing-text.md`.

## Recommended

1. Include checksum reference (`SHA256SUMS.txt`) in external release notes.
2. Keep metadata consistent with other stores for trust and support simplicity.
