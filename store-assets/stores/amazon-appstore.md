# Amazon Appstore Checklist

## Required

1. Package name: `com.rohith.nithya`
2. Use the same signing key used in previous builds.
3. Upload signed APK (`app-release-signed-v1.0.apk`).
4. Set age rating to `13+`.
5. Provide privacy policy URL: `https://nithya.fit/privacy-policy.html`.
6. Add app icon, screenshots, short description, and full description from `../common/listing-text.md`.

## Recommended

1. Keep `SHA256SUMS.txt` in your public release notes page.
2. Mention permissions purpose:
   - notifications for reminders
   - internet for sync
