# Huawei AppGallery Checklist

## Required

1. Package name: `com.rohith.nithya`
2. Keep signing key unchanged for all updates.
3. Upload signed APK (`app-release-signed-v1.0.apk`) unless AppGallery requests a different artifact.
4. Set age rating: `13+`.
5. Provide privacy policy URL: `https://nithya.fit/privacy-policy.html`.
6. Add icon, screenshots, and listing copy from `../common/listing-text.md`.

## Recommended

1. Keep a public checksum file (`SHA256SUMS.txt`) in your project release page.
2. Ensure OAuth/provider integrations are tested on Huawei devices if Google services are limited.
