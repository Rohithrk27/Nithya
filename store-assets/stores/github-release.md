# Direct GitHub Release Checklist

1. Create a new GitHub release tag (for example `v1.0.0`).
2. Upload:
   - `app-release-signed-v1.0.apk`
   - `app-release-signed-v1.0.aab`
   - `SHA256SUMS.txt`
   - `upload_certificate.pem`
   - `RELEASE_NOTES-v1.0.md`
3. Add release notes summary from `release-apk-files/RELEASE_NOTES-v1.0.md`.
4. Add privacy policy link: `https://nithya.fit/privacy-policy.html`.
5. Mention target audience: `13+`.
