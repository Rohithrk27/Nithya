# Nithya Privacy Policy

Effective date: March 4, 2026

## 1. Scope

This policy explains how Nithya collects, uses, and stores data when you use the app and website.

## 2. Data We Collect

- Account data: email, auth identifiers, and profile basics.
- Progress data: habits, quest completions, streaks, XP logs, punishments, recovery plans, and analytics summaries.
- Notification settings: reminder time, push subscription data, and notification permission status.
- Technical data: basic device/runtime diagnostics needed for app reliability and security.
- Web analytics data (web app only in production): basic page/performance metrics.
- Support/verification data: payment verification details only when you submit them.

## 3. How We Use Data

- Run core app features (login, habits, quests, profile, reminders, rankings).
- Sync progress across devices and sessions.
- Improve app stability and prevent abuse.
- Send reminder notifications only if you grant permission.
- Review voluntary donation verification submissions when provided by you.

## 4. Data Sharing and Processors

- We do not sell personal data.
- We share data only with providers needed to operate the product:
  - Supabase (authentication, database, storage, and API infrastructure).
  - Vercel Analytics and Vercel Speed Insights (web app analytics/performance metrics only).
- We may disclose data if required by law.

## 5. Donations and Payments

- UPI contributions in Nithya are optional support donations.
- Donations do not unlock digital goods, premium features, or gameplay advantages.
- If you submit a payment verification request, we store the details you submit (amount, transaction reference, optional payer/app fields, notes, timestamps, and review status).

## 6. Notifications and Permissions

Nithya requests only required permissions:

- `INTERNET`: app connectivity and sync.
- `POST_NOTIFICATIONS`: daily reminders (Android 13+ runtime consent).
- `SCHEDULE_EXACT_ALARM`: optional strict-timing reminders if enabled by the user.

## 7. Data Retention and Deletion

- Data is retained while your account is active and for operational/legal needs.
- You may request account/data deletion by contacting `itsnithyaapp@gmail.com`.

## 8. Security

- Production traffic is required over HTTPS.
- Release artifacts are signed before distribution.
- Users are encouraged to verify published SHA-256 checksums before installing APK files.

## 9. Your Choices

- You can disable reminders/notifications in app or device settings.
- You can control public profile visibility in the app.

## 10. Changes to This Policy

Policy updates will be published in this repository with an updated effective date.
